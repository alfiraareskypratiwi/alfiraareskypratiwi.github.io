Pengalaman Selama Perkuliahan di PTIK

Semester satu saya banyak diberikan tugas pemrograman menggunakan bahasa JAVA di Netbeans dan karena mulai terbiasa saya pun merasa bahwa saya menyukai bahasa JAVA. Hingga saya bertemu dengan project membuat koneksi JAVA dan MYSQL, dan dari situlah saya mulai membenci bahasa JAVA.

Di semester dua saya banyak diberikan tugas basis data berupa project dan laporannya. Meskipun project cukup banyak tapi saya tidak terlalu kepusingan karena mungkin sudah terbiasa pusing bertemu JAVA dan MYSQL di semester satu. Selain aktif di perkuliahan saya juga aktif mengikuti kepanitiaan HIMATIK-FT UNM seperti Academic Culture Orientation, Rapat Kerja, Seminar Nasional, dan ICE Skill Up. Dari kepanitiaan ini saya banyak belajar mengatur waktu, berdiskusi, dan memperbanyak relasi.
